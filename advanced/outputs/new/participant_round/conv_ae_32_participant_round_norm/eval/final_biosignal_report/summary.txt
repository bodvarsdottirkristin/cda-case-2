Interpretation guide
--------------------
ARI/NMI evaluate how similar cluster labels are to known metadata labels.
Chi-square p-values test whether cluster membership is associated with a categorical variable.
Cramer's V measures association strength: near 0 = weak, around 0.3 = moderate, 0.5+ = strong.
Mann-Whitney U tests whether feature values differ between a cluster and the remaining clusters.
Cohen's d measures the size of that difference. Larger absolute values indicate stronger discriminating features.
If --processed-file was provided, discriminating features are biosignal summaries such as HR_mean or EDA_std.
Otherwise, discriminating features are latent/PCA dimensions.
